# Security Policy

## Supported Versions

We release patches for security vulnerabilities. The following versions are currently supported:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please send an email to security@kenya-overwatch.go.ke

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

We aim to respond within 48 hours and provide a timeline for the fix.

## Security Features

- JWT-based authentication with refresh tokens
- Role-based access control (RBAC)
- Rate limiting on all endpoints
- SQL injection protection
- XSS protection
- CORS configuration
- Audit logging for all actions
- Encrypted data at rest and in transit

## Compliance

This system is designed to meet:
- Kenya Data Protection Act 2019
- GDPR requirements
- Government security standards

## Updates

Security updates will be announced in the release notes.
